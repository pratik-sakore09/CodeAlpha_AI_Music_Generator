# CodeAlpha_AI_Music_Generator
Deep Learning based AI Music Generator using LSTM, TensorFlow, Streamlit and Music21.
# 🎵 AI Music Generator

## CodeAlpha AI Internship - Task 3

A Deep Learning application that generates MIDI music using LSTM neural networks.

## Features

- Upload MIDI Files
- Train LSTM Model
- Generate AI Music
- Interactive Piano Roll
- Streamlit Dashboard
- Music Analysis
- MIDI Export

## Technologies

- Python
- TensorFlow
- Keras
- Streamlit
- Music21
- NumPy
- Pandas
- Matplotlib

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run music.py
```
